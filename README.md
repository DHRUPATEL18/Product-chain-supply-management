# Pragmanx Onelife Distributor

A web-based distributor management system designed to handle product distribution, inventory, and user management efficiently.

---

## 🚀 Features

* 📦 Product Management
* 🧾 Order & Distribution Tracking
* 👤 User Authentication & Authorization
* 🗄️ MySQL Database Integration
* ⚡ Fast and responsive interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL

---

## 📂 Project Structure

/project-root
│── /assets        # CSS, JS, images
│── /includes      # Reusable PHP files
│── /pages         # Main pages
│── config.php     # Database configuration
│── index.php      # Entry point

---

## ⚙️ Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/pragmanx_onelife_distributor.git
   ```

2. Move project to server directory:

   * XAMPP → `htdocs`
   * WAMP → `www`

3. Import the database:

   * Open phpMyAdmin
   * Create a new database
   * Import the `.sql` file

4. Configure database connection:

   ```php
   $host = "localhost";
   $user = "root";
   $password = "";
   $database = "your_db_name";
   ```

5. Run the project:

   ```
   http://localhost/pragmanx_onelife_distributor
   ```
